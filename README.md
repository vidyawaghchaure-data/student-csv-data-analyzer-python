# 🎓 Student CSV Data Analyzer

A simple **Python-based Student CSV Data Analyzer** that manages student records using CSV file handling. This project demonstrates Python fundamentals such as file handling, functions, loops, conditional statements, and menu-driven programming.

## 📌 Features

* 👀 View All Students
* ➕ Add Student
* 🔍 Search Student
* 🗑️ Delete Student
* 📊 Calculate Average Marks
* 🏆 Find Highest Marks
* 👨‍🎓 Count Total Students
* 🚪 Exit Program

## 🛠️ Technologies Used

* Python
* CSV Module


## ▶️ How to Run

1. Download or clone the repository.
2. Open the project in VS Code.
3. Make sure `student.csv` is in the same folder as `main.py`.
4. Run the program:

```bash
python main.py
```

## 📚 Concepts Covered

* CSV File Handling
* Functions
* Loops
* If-Else Statements
* Exception Handling
* Menu-Driven Programming

## 🎯 Learning Outcome

This project helps beginners understand how to store, read, search, analyze, and manage data using CSV files in Python.
